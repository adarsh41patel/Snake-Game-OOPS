Project: Snake Game using HTML, CSS, and JavaScript

Developed a classic Snake game using object-oriented programming concepts in HTML, CSS, and JavaScript. Implemented features such as HTML5 canvas, game loop, pen, and event listeners to create an interactive gaming experience. The game is rendered on a black background using the HTML5 canvas element, providing a visually appealing interface. The snake's movement is controlled by the arrow keys, and the objective is to eat the food and grow longer without colliding with the walls or its own body. Utilized JavaScript to manage the game loop, update the snake's position, detect collisions, and handle user input. Demonstrated proficiency in front-end development, problem-solving, and application of object-oriented programming principles.
